
🚀 CRM Sales Dashboard | Power BI
📌 Project Overview

This project showcases a CRM Sales Pipeline Dashboard built in Power BI to help sales teams monitor pipeline performance, revenue forecasting, account growth, and sales activity tracking.

The dashboard provides interactive insights into opportunities, accounts, sales stages, win rates, and revenue trends to support data-driven decision making.

🎯 Business Objective

The objective of this dashboard is to:

Track sales pipeline performance
Monitor open opportunities & revenue
Analyze win/loss trends
Measure sales rep performance
Identify high-value customers and growth opportunities
Improve forecasting and sales visibility

📊 Dashboard Features
Executive Overview
Total Pipeline Revenue
Open Opportunities Count
Closed Won / Closed Lost Metrics
Win Rate %
Revenue Trend Analysis
Sales Pipeline Analysis
Opportunity Stage Distribution
Pipeline by Sales Rep
Revenue by Deal Stage
Opportunity Aging Analysis
Forecast vs Actual Revenue
Account & Customer Insights
Top Customers by Revenue
Industry / Sector Analysis
Customer Segmentation
Regional Performance Overview
Performance Monitoring
Sales Representative KPIs
Monthly Sales Performance
Conversion Funnel Analysis
Sales Cycle Duration Tracking

🛠️ Tech Stack
Power BI Desktop
DAX
Power Query (M)
Data Modeling
Interactive Visualizations
