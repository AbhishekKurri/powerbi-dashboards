**👥 HR Dashboard | Power BI**
**📌 Project Overview**

- The HR Dashboard is an interactive Power BI reporting solution designed to provide comprehensive workforce insights across employee demographics, hiring trends, attrition, performance, and organizational metrics.

- The dashboard helps HR teams and business leaders monitor workforce health, improve employee retention, optimize hiring strategies, and support people-focused decision making.


**🎯 Business Objective**

The objective of this dashboard is to:

- Monitor workforce composition and employee distribution
- Track hiring and attrition trends
- Analyze employee demographics
- Measure retention and turnover performance
- Evaluate department-level workforce metrics
- Support strategic HR planning and decision making


**📊 Dashboard Features**
- Executive HR Overview
- Total Employees
- Active Employees
- Attrition Rate %
- Hiring Count
- Average Employee Tenure
- Gender Diversity Metrics
- Workforce Demographics
- Employee Distribution by Department
- Age Group Analysis
- Gender Distribution
- Education Background Analysis
- Job Role Breakdown
- Attrition & Retention Analysis
- Employee Attrition Trends
- Department-wise Turnover Analysis
- Attrition by Age / Job Role / Experience
- Retention Performance Tracking
- Exit Pattern Analysis
- Hiring & Recruitment Insights
- Hiring Trend Analysis
- Recruitment Source Performance
- New Joiners Analysis
- Department Hiring Metrics
- Employee Performance Analysis
- Performance Rating Distribution
- Employee Satisfaction Metrics
- Tenure vs Performance Insights
- Promotion & Career Growth Tracking


**🛠️ Tech Stack**
- Power BI Desktop
- DAX
- Power Query (M Language)
- Data Modeling
- Interactive Data Visualization
